    - (void) build(paths = [], callback = Utils.printCallback) (bound)

    Build CoffeeScript to JS on an array of files, directory paths.

    Note: The paths parameter takes an array of either string source files or object source / destination object pairs.

    Note: The coffee binary must be installed separately and available to a shell invocation.

    Examples:

    paths 

    paths = [
  "foo.coffee"
  { "src_dir": "lib_dir" }
  "bar.coffee"
]

    Parameters:

    • paths (Array) — Array of file and source / dest dirs.
    • callback (Function) — Callback on process end (printCallback).
